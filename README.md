## Amsterdam Golang Meetup 02-19

### Rune Magic in Go 

This repo contains the sources used in the "Rune Magic in Go" talk.

### Requirements

- go 1.11

- dep

Install dep:

    $ brew install dep
or    
  
    $ go get -u github.com/golang/dep/cmd/dep
